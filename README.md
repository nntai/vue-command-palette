# Command Palette Component Demo

This is a demo for Vue's Command Palette component.

## Installation

```bash
yarn install
```

## Running on local

```bash
yarn dev
```

Now this demo app will run on your [localhost:3000](http://localhost:3000/)

## Tech stack

- Vue 3
- Vite
- TypeScript
- Jest (adding)
